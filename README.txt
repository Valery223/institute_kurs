# Проект: Python Application

## Описание
Этот проект представляет собой оптимизация расписания автобусов с использованием генетического алгоритма.

---

## Клонирование проекта
Для начала работы склонируйте репозиторий на свой локальный компьютер:
```bash
git clone https://github.com/Valery223/institute_kurs
```

Перейдите в директорию проекта:
```bash
cd your-repo-name
```

---

## Установка зависимостей
Рекомендуется использовать виртуальное окружение для изоляции зависимостей проекта.

### 1. Создайте виртуальное окружение
```bash
python -m venv venv
```

### 2. Активируйте виртуальное окружение
- **Windows:**
  ```bash
  venv\Scripts\activate
  ```
- **macOS/Linux:**
  ```bash
  source venv/bin/activate
  ```

### 3. Установите зависимости
После активации виртуального окружения установите необходимые зависимости из файла `requirements.txt`:
```bash
pip install -r requirements.txt
```

---

## Запуск проекта
Для запуска проекта выполните следующую команду:
```bash
python main.py
```

---

## Требования
- Python версии 3
- Дополнительные зависимости указаны в `requirements.txt`.
